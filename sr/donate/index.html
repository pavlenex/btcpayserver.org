Podržite BTCPay Projekat

BTCPay Server je ponosno besplatan i otvorenog tipa, građen i održavan od strane zajednice strastvenih dobrovoljaca širom sveta.

Korporativne Donacije

Poslovni entiteti koji žele da podrže budućnost projekta, treba da posete sajt BTCPay Server Fondacije i postanu zvanične pristalice.
BTCPay Server Fondacija  

Individualne Donacije

Ukoliko niste posloni entitet, možete podržati pojedinačno dobrovoljce ispod. Donacije idu direktno dobrovoljcima, na serverima koje oni hostuju i nisu deo BTCPay Server Fondacije.
